NRARFCN → Tần số (MHz)
N < 600 000    → F = N × 0.005
N ≤ 2 016 666 → F = 3000 + (N − 600 000) × 0.015
N > 2 016 666 → F = 24 250.08 + (N − 2 016 667) × 0.06
Tần số → NRARFCN
F < 3 000   → N = round(F / 0.005)
F < 24 250 → N = round((F − 3000) / 0.015 + 600 000)
F ≥ 24 250 → N = round((F − 24 250.08) / 0.06 + 2 016 667)
GSCN → F_SSB (MHz)
GSCN < 7 499   → F = N×1.2 + M×0.05 (N=GSCN÷3, M∈{1,3,5})
GSCN ≤ 22 255 → F = 3 000 + (GSCN − 7 499) × 1.44
GSCN > 22 255 → F = 24 250.08 + (GSCN − 22 256) × 17.28
